# Google-crawler-and-Automated-Gmail-Using-SMTP
🚀 Digital Marketing Automation using Python

A complete end-to-end automation pipeline for digital marketing that:

Crawls the web for business leads
Extracts contact email addresses
Sends automated outreach emails

Built using Python with a modular and scalable architecture.

📌 Project Overview

Manual lead generation and outreach is time-consuming and inefficient.
This project automates the entire workflow:

🔍 Search for businesses using custom queries
🌐 Visit websites and extract email addresses
📧 Send professional outreach emails automatically

👉 All steps are executed using a single command pipeline

⚙️ Technologies Used
Python
Requests
BeautifulSoup
Regular Expressions (Regex)
SMTP (Gmail)
python-dotenv
VS Code

🔄 Workflow
User defines search queries
Crawler fetches search results
System visits each website
Emails are extracted using regex
Data is saved in CSV
Emails are sent using Gmail SMTP

🧠 Key Features
Automated lead generation
Email extraction from websites
Bulk email sending
Modular code design
Secure credential handling using .env
Logging system for debugging

✔ Generated 30+ business leads
✔ Extracted real email addresses
✔ Successfully sent automated emails
